Présentation de Joomla

Cette présentation à été tiré du site : sylvie-ceci.info

Joomla est une plateforme qui propose une console d'administration, conviviale, intuitive, accessible à tous. Pas besoin d'être un expert en informatique pour l'utiliser.

Cette console est sécurisée, on y accéde à l'aide d'un login et d'un mot de passe. Cette console d'administration est composée de boutons aux intitulés très explicite comme :

  • Ajouter un article
  • Gérer les médias

Joomla et ses fonctionnalités

Joomla propose au départ, dans sa forme basique un certain nombre de fonctionnalités natives comme :

  • La création de formulaires de contact
  • La possibilités de créer des redirections
  • Les URL reformulées
  • La recherche
  • Un captcha
  • Fils d'actualité
  • Messagerie privée

Mais ce qui est interessant, c'est que l'on peut en ajouter d'autres. Vous avez un projet Internet ? Ajoutons des fonctionnalités ..
Vous souhaitez réaliser :

  • Un blog avec la possibilité de partager vos articles, de laisser des commentaires, des appréciations ..
  • Le site Internet de votre entreprise presentant ses activités, ses produits sous forme de galeries d'images, de fiches produits
  • Un site de vente avec paiement en ligne ..
  • Un site multiligue ...

Joomla s'adapte à vos demandes.

De nombreuses extensions existent appelées composants, modules, plugIn. Sur le logiciel de base, il suffit de les installer et de les configurer.

Pour mieux comprendre ces termes : Composants, modules et plugIn :

  • Un composant est une fonctionnalité, une petite application intégrée à votre site Internet.
  • Un module est un "bloc" que l'on retrouve souvent à droite ou à gauche du contenu. Il est souvent lié à un composant. Il affiche les images d'une galerie, l'inscription à une NewsLetter ...
  • Un plug In est un petit "morceau de code" qui permet l'affichage d'images dans un article, par exemple.

Les fonctionnalités de votre futur site

Un exemple très simple de composant indispensable pour simplifier la création d'articles : l'éditeur de texte et d'images JCE. C'est un composant très interessant que j'installe systèmatiquement sur les sites Internet de mes clients.
Ce composant vous permet d'être aussi à l'aise dans la console d'administration de Joomla que devant votre logiciel WORD.

Il vous permet de traiter votre texte :

  • saisie, copier/coller
  • modification de polices, de couleurs, de taille des caractères...

Il vous permet de traiter vos photos :

  • en les redimentionnant
  • en les insérant facilement

Et ce ne sont que quelques exemples..

D'autres fonctionnalités indispensables

Pour augmenter la sécurité de Joomla : Admin Tools

Son utilité :

  • Notifie les mises à jour Joomla
  • Améliore la sécurité de Joomla : mots de passe, préfixe des tables de Joomla
  • Propose des outils de maintenance : configuration des permissions, nettoyage du dossier temporaire ...

Pour sauvegarder Joomla et sa base de données : Akeeba Backup

Son utilité :

  • Sauvegarde et restaure votre site Internet

Pour optimiser le temps de chargement : JCH Optimise

Son utilité :

  • Compresse les fichiers Javascript et CSS, active la compression GZIP, minifie les fichiers javascript, CSS, HTML, etc.

Pour améliorer encore la sécurité de Joomla : jHackGuard

Son utilité :

  • Protège votre site contre les tentatives d'injections SQL
